#### 3.	Створити наступну матрицю за допомогою matrix, та за допомогою cbind або rbind
| 0.5 | 1.3 | 3.5 |
|---|---|---|
| 3.9 | 131 | 2.8 |
| 0   | 2.2 | 4.6 |
| 2   | 7   | 5.1 |

```{r}
A <- matrix(c(0.5, 3.9, 0, 2, 1.3, 131, 2.2, 7, 3.5, 2.8, 4.6, 5.1), nrow = 4, ncol = 3)
A
```

Результат:

| 0.5 | 1.3 | 3.5 |
|---|---|---|
| 3.9 | 131 | 2.8 |
| 0   | 2.2 | 4.6 |
| 2   | 7   | 5.1 |

```{r}
a <- c(0.5, 3.9, 0, 2)
b <- c(1.3, 131, 2.2, 7)
c <- c(3.5, 2.8, 4.6, 5.1)
cbind_matrix <- cbind(a, b, c)
cbind_matrix
```
Результат:

| 0.5 | 1.3 | 3.5 |
|---|---|---|
| 3.9 | 131 | 2.8 |
| 0   | 2.2 | 4.6 |
| 2   | 7   | 5.1 |

```{r}
a <- c(0.5, 1.3, 3.5)
b <- c(3.9, 131, 2.8)
c <- c(0, 2.2, 4.6)
d <- c(2, 7, 5.1)
rbind_matrix = rbind(a, b, c, d)
rbind_matrix
```
Результат:

| 0.5 | 1.3 | 3.5 |
|---|---|---|
| 3.9 | 131 | 2.8 |
| 0   | 2.2 | 4.6 |
| 2   | 7   | 5.1 |
